This is your org's public knowledge base. Edit it. The site you're reading lives in markdown files at `docs/*.md` in [the repo](https://git.publicinterestalberta.org/pia-starter-stack) — change a file, save, refresh; the site rebuilds.

## What's published here

- [Research](research.md) — what we've found, what we're learning.
- [Subscribe](subscribe.md) — how to get our updates by email.

## How to edit this site

Three options, pick the one that matches how you work:

1. **In a text editor on your laptop.** Open the markdown file under `docs/`, edit, save. The running container will rebuild automatically.
2. **Through Git.** Clone the repo, edit, commit, push. Pull on the server, the site rebuilds.
3. **Hand it off to a generalist.** Markdown is plain text — anyone on your team can write it.

## Why this stack

This site, plus a newsletter system at [our subscribe page](subscribe.md), runs entirely on free, open-source software you control. Nothing is rented from Substack, Squarespace, or Mailchimp. The whole thing is one `docker-compose.yml` file you can fork, modify, and host anywhere.

If you're seeing this page, your starter stack is working. Now make it yours.
